# Gluconet !
<img src="res/logo_site.png" alt="Gluconet Logo" width="100"/>

Gluconet est un site web mettant à disposition un outil de traçage du taux de glucose Pour des personnes atteintes de diabète. 

Le site permet aux utilisateurs de suivre et d'analyser leurs niveaux de glucose sanguin au fil du temps, offrant des graphiques interactifs et des rapports personnalisés pour mieux gérer leur condition.

Le site présente égalemment d'autres fonctionnalités telles que :
- La mise en place de rappels de prise de médicaments, 
- La possibilité de communiquer avec votre médecin traitant via une messagerie,
- La possibilité d'établir un journal alimentaire pour suivre l'impact de l'alimentation sur les niveaux de glucose.
